**Design Style Prompt**

Create a dark, high clarity command interface for a real time disaster intelligence platform.

Visual tone should feel calm, precise, and mission critical. Think control room energy. Not loud. Not flashy. Confident and focused.

Use a deep gradient background with subtle atmospheric blending of navy, forest green, and muted teal. The gradient should feel smooth and immersive, slightly blurred, almost like a radar screen in low light.

Typography must be clean, modern, and minimal. Large bold headline with generous spacing. Secondary text should be soft gray with reduced contrast but still readable. Avoid clutter. Use strong hierarchy.

Cards should have:
• Soft glass morphism feel
• Subtle border glow
• Rounded corners
• Low opacity background
• Gentle shadow for depth

Buttons should:
• Use soft neon green or muted lime accent
• Be pill shaped
• Have minimal hover glow
• Feel tactical and intentional

Spacing should be generous. The layout must breathe. No visual noise.

Overall feeling:
Minimalist
High trust
AI powered
Command center
Real time intelligence
Calm in chaos

---

**Language Style Prompt**

Copy must feel sharp, decisive, and operational.

Short sentences.
Clear impact.
No fluff.
No hype.
No marketing noise.

Use structured language that sounds like a system reporting status.

Examples of tone:
Clarity in chaos.
Signal detected.
Confidence stable.
Pressure rising.
Monitoring active.

Microcopy should feel like telemetry, not sales writing.

Avoid:
Exclamation marks
Buzzwords
Long paragraphs
Emotional exaggeration

Voice characteristics:
Authoritative
Technical but human
Grounded in urgency
Built for responders
